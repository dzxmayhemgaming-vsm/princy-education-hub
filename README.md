# Princy Education Hub (Rajasthan Exam Prep Portal)

स्वागत है! **Princy Education Hub** राजस्थान की सभी प्रतियोगी परीक्षाओं (जैसे RAS, REET, Rajasthan Police SI, Patwari, VDO, CET, LDC आदि) की तैयारी के लिए एक अति-आधुनिक और प्रीमियम डिजिटल स्टडी पोर्टल है।

यह वेबसाइट पूरी तरह से **Single Page Application (SPA)** मॉडल पर आधारित है जिसे **HTML5, CSS3, और Modern JavaScript** का उपयोग करके डिज़ाइन किया गया है। यह अत्यंत तेज़, उत्तरदायी (Responsive), और बिना किसी बाहरी लाइब्रेरी या सेटअप के आपके कंप्यूटर पर तुरंत चलने के लिए तैयार है।

---

## 🌟 मुख्य विशेषताएं (Core Features)

1. **स्टनिंग डैशबोर्ड (Dashboard & Search)**: परीक्षाओं के लिए आकर्षक हीरो सेक्शन और लाइव सर्च फिल्टर।
2. **इंटरैक्टिव CBT मॉक टेस्ट सिम्युलेटर (CBT Simulator)**:
   - बिल्कुल वास्तविक बोर्ड परीक्षाओं जैसा कंप्यूटर आधारित टेस्ट (CBT) इंटरफ़ेस।
   - टाइमर कंट्रोल, स्टेटस पैलेट (Answered, Not Answered, Marked, Not Visited)।
   - वास्तविक परीक्षा मार्किंग स्कीम (नेगेटिव मार्किंग के साथ) पर आधारित तत्काल ऑटो-ग्रेडिंग।
   - द्विभाषी (Hindi/English) शार्ट एक्सप्लेनेशन और हल के साथ विस्तृत स्कोरकार्ड।
3. **वर्चुअल क्लासरूम और संपूर्ण कोर्सेज (Virtual Classroom)**:
   - व्यवस्थित अध्यायों (Chapters) और प्लेलिस्ट लेक्चर व्यूअर के साथ कोर्सेज।
   - लेक्चर वीडियो प्लेबैक सिम्युलेटर और प्रगति (Progress Tracker) ट्रैकर।
4. **हस्तलिखित पीडीएफ नोट्स रिपोजिटरी (PDF Notes Vault)**:
   - विषयवार (Subject-wise) लाइव फिल्टरिंग और इंस्टेंट सर्च।
   - इन-ऐप (In-App) पीडीएफ मॉक रीडर जिससे छात्र बिना डाउनलोड किए वेबसाइट पर ही नोट्स पढ़ सकते हैं।
   - सिमुलेटेड ऑफ़लाइन डाउनलोड ट्रिगर।
5. **दैनिक राजस्थान GK बूस्टर चैलेंज (Daily GK Quiz)**:
   - छात्रों को आकर्षित करने के लिए दैनिक 3-प्रश्नों का गेमीफाइड क्विज़ गेम।
   - सही/गलत उत्तरों पर तत्काल विजुअल फीडबैक, स्कोर कैलकुलेशन और शार्ट एक्सप्लेनेशन।
6. **सुंदर लाइट/डार्क थीम (Adaptive Theme)**:
   - आँखों की सुरक्षा के लिएpersisted लाइट और डार्क मोड टॉगल।

## 🚀 लाइव कैसे करें (How to Host it Live)

चूंकि आपके कंप्यूटर पर Node.js या Git इंस्टॉल नहीं है, इसलिए इसे लाइव करने का सबसे आसान, सुरक्षित और तेज़ तरीका **Vercel Drag-and-Drop** है।

### विधि 1: Vercel Drag-and-Drop (अनुशंसित - सिर्फ 30 सेकंड में!)

1. सबसे पहले अपने वेब ब्राउज़र में [Vercel Upload Portal](https://vercel.com/new) पर जाएं।
2. यदि आपका अकाउंट नहीं है, तो अपने Email या Google/GitHub से साइन-इन करें।
3. वहाँ आपको एक **"Drag and Drop your folder here"** का बॉक्स दिखाई देगा।
4. अपने कंप्यूटर पर इस प्रोजेक्ट के फोल्डर (`Princy Education Hub`) को सीधे ड्रैग करके उस बॉक्स में छोड़ (Drop) दें।
5. Vercel आपके कोड को पढ़ेगा और **Deploys** बटन पर क्लिक करते ही 5 सेकंड में आपकी वेबसाइट को एक शानदार **फ्री लाइव डोमेन URL** (जैसे: `princy-education-hub.vercel.app`) पर होस्ट कर देगा!

### विधि 2: GitHub + Vercel (स्वचालित अपडेट्स के लिए)

यदि आपके पास Git इंस्टॉल है या आप किसी दूसरे सिस्टम से इसे पुश करना चाहते हैं:
1. [GitHub](https://github.com) पर एक नया रिपोजिटरी (उदा. `princy-education-hub`) बनाएं।
2. इस फोल्डर के अंदर टर्मिनल खोलें और निम्नलिखित कमांड चलाएं:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of Princy Education Hub"
   git branch -M main
   git remote add origin <your-github-repo-url>
   git push -u origin main
   ```
3. अब [Vercel Dashboard](https://vercel.com/dashboard) पर जाएं, **"Add New Project"** पर क्लिक करें।
4. अपने GitHub अकाउंट को कनेक्ट करें और इस रिपोजिटरी को इम्पोर्ट करें।
5. **Deploy** पर क्लिक करें। अब भविष्य में जब भी आप GitHub पर कोई कोड अपडेट करेंगे, Vercel उसे आटोमेटिक लाइव कर देगा!

## 🛠️ फ़ाइल संरचना (File Directory Structure)

- `index.html` - मुख्य लेआउट, नेविगेशन बार, दराज और रूट कंटेनर।
- `assets/css/`
  - `style.css` - थीम टोकन (रंग, फोंट), ग्लासमोर्फिज्म शैलियाँ और स्क्रॉलबार।
  - `home.css` - डैशबोर्ड, हीरो बैनर, और स्टैट्स ग्रिड।
  - `test-portal.css` - मॉक टेस्ट CBT सिम्युलेटर और रिज़ल्ट्स।
  - `courses.css` - कोर्स प्लेलिस्ट, प्लेयर और अकोर्डियन।
  - `pdf-vault.css` - पीडीएफ फाइल्स सूची और इन-ऐप रीडर modal।
- `assets/js/`
  - `exams-data.js` - हमारी स्थानीय डेटाबेस फ़ाइल (मॉक टेस्ट पेपर्स, कोर्सेज, पीडीएफ लिंक्स)।
  - `app.js` - मुख्य SPA राउटर, मोबाइल दराज इवेंट्स, थीम स्विचर।
  - `test-portal.js` - टेस्ट शुरू/खत्म, ग्रेडिंग एल्गोरिथम, व्याख्या।
  - `courses.js` - क्लासरूम लेक्चर एक्टिवेशन, प्रोग्रेस ट्रैकर।
  - `pdf-vault.js` - पीडीएफ नोट्स लोड, सर्च, और इन-ऐप मॉक रीडर।
  - `gk-quiz.js` - सामान्य ज्ञान बूस्टर क्विज़।
